Euroscores

This project revolves around EuroScores, a sleek and interactive PWA developed in React + Vite to keep track of live scores and statistics during the EURO 2024 tournament. By leveraging a robust API, EuroScores provides users with real-time updates, player stats, and match highlights in an intuitive and well-organized interface. The platform is designed with a focus on simplicity and accessibility, ensuring fans can effortlessly stay informed about their favorite teams and matches. This project includes comprehensive UI/UX design mockups crafted in Figma, along with a fully functional prototype that showcases the seamless integration of live data into the app's dynamic dashboard.

Technologies used

Figma Illustrator Adobe XD React Vite Tailwind CSS
